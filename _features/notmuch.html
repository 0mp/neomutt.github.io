 Notmuch Patch

Email search engine

1. Patch

To check if Mutt supports “Notmuch”, look for “+USE_NOTMUCH” in the mutt version. See: Section 1.1, “Enabling/Disabling Features”.

Dependencies:

  • mutt-1.5.24

  • sidebar patch

  • quasi-delete patch

  • index-color patch

  • Notmuch libraries

This patch is part of the NeoMutt Project.
