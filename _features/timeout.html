 Timeout Patch

Run a command periodically

1. Patch

To check if Mutt supports “ timeout”, look for “ patch-timeout”in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.7.0

This patch is part of the NeoMutt Project.

2. Introduction

This patch implements a new hook that is called periodically when Mutt checks for new mail. This hook is called every $timeoutseconds.

3. Commands

timeout-hook . MUTT-COMMAND

4. Muttrc

# This is a complete list of timeout configuration.

# --------------------------------------------------------------------------
# COMMANDS - shown with an example arguments
# --------------------------------------------------------------------------

# After $timeout seconds of inactivity, run this mutt command
timeout-hook . 'exec sync-mailbox'

# vim: syntax=muttrc
