 Virtual Unmailbox Patch

Make the file browser behave

1. Patch

To check if Mutt supports “ virt-un”, look for “ patch-virt-un”in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.7.0

This patch is part of the NeoMutt Project.

2. Introduction

This patch helps users of NotMuch who have multiple mail accounts. When you switch between accounts the virtual-unmailboxes will clean up the NotMuch queries that you no longer need.
