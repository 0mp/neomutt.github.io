 compose-sender Feature

Send new mail to the sender of the current mail

1. Support

To check if Mutt supports “ compose-sender”, look for “ patch-compose-sender”in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.7.0

This feature is part of the NeoMutt Project.

2. Introduction

The compose-sender patch adds a new command to start composing an email the sender of the current message. It works on tagged messages too, sending one email to all of the senders of the tagged messages.

3. Functions

compose-sender adds the following functions to Mutt. By default, none of them are bound to keys.

Table 6.1. compose-sender Functions

Menus Function Description
index,pager <compose-sender> compose a new email to the sender of the current email

4. Muttrc

# Example NeoMutt config file for the compose-sender feature.

# --------------------------------------------------------------------------

# --------------------------------------------------------------------------
