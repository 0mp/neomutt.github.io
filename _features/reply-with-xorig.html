 Reply With X-Original-To Patch

Adds a configuration variable to allow direct reply to a mail using X-Original-To header as a From: in the answer.

1. Patch

To check if Mutt supports “ Reply With X-Original-To”, look for “ patch-reply-with-xorig”in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.7.0

This patch is part of the NeoMutt Project.

2. Introduction

Adds a reply_with_xorig for mutt configuration files. If enabled, allows to reply to an email using the email address in the first X-Original-To: header of a mail as the From: header of the answer.

3. Variables

Table 6.22. Reply With X-Original-To Variables

Name Type Default
reply_with_xorig Boolean no

4. Muttrc

# This is a complete list of reply-with-xorig-related configuration.
 
# --------------------------------------------------------------------------
# VARIABLES - shown with their default values
# --------------------------------------------------------------------------
 
# Use X-Original-To header to reply when reverse is disabled or no alternate
# is found.
set reply_with_xorig = "yes"
