 Limit-Current-Thread Patch

Focus on one Email Thread

1. Patch

To check if Mutt supports “ limit-current-thread”, look for “ patch-limit-current-thread”in the mutt version. See: Section 1.2, “Mutt Patches”.

Dependencies:

  • mutt-1.7.0

This patch is part of the NeoMutt Project.

2. Introduction

This patch adds a new way of using the Limit Command. The <limit-current-thread>function restricts the view to just the current thread. Setting the limit (the lkey) to “ all”will restore the full email list.

3. Functions

Limit-current-thread adds the following function to Mutt. By default, it is not bound to a key.

Table 6.13. Limit-Current-Thread Functions

Menus Function Description
index <limit-current-thread> Limit view to current thread

4. Muttrc

# Example Mutt config file for the 'limit-current-thread' patch.
 
# Limit view to current thread
bind index <esc>L limit-current-thread
